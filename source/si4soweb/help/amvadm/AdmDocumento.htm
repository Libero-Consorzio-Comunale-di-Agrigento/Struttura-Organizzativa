Gestione dei Documenti

 

Definizione

Tutte le pagine contenute nel Portale possono essere inserite, aggiornate o revisionate con le funzionalità disponibili al’Amministratore del portale o al personale delegato all’amministrazione dei contenuti.

I documenti introdotti subiscono il processo di approvazione che percorre gli stati di :

·      Redazione

·      Verifica

·      Approvazione.

 

Proprietà

 

Questa pagina, visualizzata sia in caso di inserimento di un nuovo documento sia in caso di aggiornamento di un documento esistente, permette di attribuire le caratteristiche al documento contenuto nel portale.

Dati

Titolo

Titolo del documento.

Revisione

Numero di revisione del documento.

Stato

Stato in cui si trova il documento.

Inserimento

Data e nome dell’autore del documento.

Aggiornamento

Data e nome dell’utente che ha modificato il documento.

Cronologia

Cronologia delle operazioni eseguite sul documento.

Note

Eventuali annotazioni apportate nel proccesso di approvazione del documento.

Area di Accesso

Area di appartenenza del documento.

Da questo informazione dipende la visibilità del documento per gli utenti del portale. Il documento sarà visibile infatti agli utenti che appartengono a gruppi con diritto di accesso almeno in lettura sull'area selezionata. (Amministratore Diritti di Accesso)

Tipologia

Tipologia di appartenenza del documento. (Amministratore Tipologie)

Categoria

Categoria di appartenenza del documento.

Argomento

Argomento di interesse del documento.

Sezione

Permette la suddivisione dei contenuti del portale nell’ambito di una struttura gerarchica.

Rilevanza

Rilevanza del documento.

Le rilevanze con importanza "HeadLine" e "HomePage" danno la visibilità nella pagina principale. (Amministratore Rilevanze)

Data riferimento

La data di riferimento del documento, preimpostata alla data corrente, può essere particolarmente significativa per gli eventi che hanno un riferiemnto nel tempo.

Va osservato che questa data viene utilizzata per l’ordinamento dei documenti negli elenchi.

Pubblicazione

Periodo di pubblicazione del documento nel portale.

Se il documento è in uso sarà visibile solo nel periodo compreso fra le date indicate.

Se la data di fine non viene indicata il documento sarà visibile finchè non viene posto fuori uso.

 

Contenuto

 

Questa pagina, visualizzata sia in caso di inserimento di un nuovo documento sia in caso di aggiornamento di un documento esistente, permette di introdurre il contenuto proprio della pagina, inserendo un testo e/o un file da cui prelevare le informazioni ed aggiungere eventuali allegati.

Dati

 

 

 

Tipo contenuto

Tipo di informazione contenuta nel documento:

Testo/File html

 Il documento contiene un testo e/o il collegamento a un file di tipo html.

Link Il documento rappresenta un collegamento ad un URL.

Modello inserimento

 Il documento è il riferimento per la registrazione di una Richiesta on-line basata su un modello di inserimento.

Testo/Abstract

Spazio per inserire il testo o il riassunto breve del contenuto del documento.

Il testo può essere composto anche con sintassi HTML. A fianco al testo è disponibile una funzione particolare (Modifica testo) per modificare il contenuto.

File html

File in formato HTML usato per comporre o completare il testo del documento; è utilizzato quando il tipo contenuto è Testo/File html.

URL

Collegamento ad un indirizzo internet utilizzato quando il tipo contenuto è Link.

Immagine

File da cui prelefare l’immagine utilizzabile nell’identificazione del documento.

Modello Richiesta

Modello prelevato dalla gestione documentale utilizzabile come modello per comporre la richiesta on-line.

Visibile quando il tipo contenuto è Modello inserimento.

Modello Approvazione

Modello prelevato dalla gestione documentale da utilizzare nel caso sia necessario un modello specifico per l’approvazione.

Visibile quando il tipo contenuto è Modello inserimento.

Iter Approvazione

Processo di approvazione che deve subire il documento:

-  Verifica e Approvazione;

-  Solo Approvazione;

-  Senza Approvazione.

Visibile quando il tipo contenuto è Modello inserimento.

Modalità Inoltro

Modalità con cui la richiesta viene inoltrata al processo di approvazione:

Immediata inoltrata immediatamente.

da Pagina inoltrata solo su indicazione dell’autore.

da Modello inoltrata con la funzionalità specifica del modello.

Visibile quando il tipo contenuto è Modello inserimento.

Allegati

Espone i file già allegati al documento.

Inserisci Allegato

Consente l’Upload di un fileda inserire come allegato al documento.

L’inserimento di un allegato può essere eseguito soltanto in fase di aggiornamento.

Elimina Allegato

Consente l’eliminazione di file già allegati al documento.

 

Funzionalità Particolari

Permette di modificare il testo con un editor HTML.

Consente di selezionare un file tra quelli presenti sul Server caricandolo con una funzione di Upload dal proprio client.

Consente di selezionare un’immagine tra quelle presenti sul Server caricandolo con una funzione di Upload dal proprio client.

 

Creazione nuova Revisione

 

Questa pagina viene visualizzata nel caso in cui venga richiesta la creazione di una revisione al contenuto di un documento in uso.

In questo caso il documento viene messo in stato "in Revisione" e il nuovo documento considerato in stato "Redazione".

All’approvazione definitiva di questa revisione il documento precedente viene messo in stato "Fuori uso".
